Edit

remove

文件删除

syberh.fileManager.remove(Object object)

参数

Object object

属性类型默认值必填描述
srcPathString是源路径
recursiveint是递归删除 0：否，1：是
successfunction否

object.success 回调函数参数

参数

属性类型必填描述
resultboolean是返回 true, 表示执行成功

示例代码

syberh.fileManager.remove({
    srcPath: '/home/user/22',
    recursive: 0,
    success: function(res) {
        console.info(res)
    }
})
